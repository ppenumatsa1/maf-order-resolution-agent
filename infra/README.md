# Infrastructure Scaffolding

This directory contains deployment scaffolding for planned Azure hosting paths.

- `azure-apphosted/`: Azure app-hosted runtime path using the current deterministic/MAF SDK backend mode.
- `foundry-hosted/`: Future Foundry-hosted runtime path scaffold (`WORKFLOW_MODE=foundry_hosted`).

Both paths are additive scaffolds only; they do not change existing local Docker/Make workflows.
